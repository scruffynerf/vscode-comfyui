- `comfyui-easy-use` — [ComfyUI-Easy-Use](https://github.com/yolain/ComfyUI-Easy-Use) *(2 nodes)*: imagesCountInDirectory, isFileExist
- `mikey_nodes` — [mikey_nodes](https://github.com/bash-j/mikey_nodes) *(2 nodes)*: FileNamePrefix, FileNamePrefixDateDirFirst
- `comfyui_controlnet_aux` — [comfyui_controlnet_aux](https://github.com/Fannovel16/comfyui_controlnet_aux): SavePoseKpsAsJsonFile

---

To install a pack from this list:
```bash
cd {installDir}
uv pip install --extra-index-url https://nodes.appmana.com/simple/ <id>
```
Then restart the ComfyUI server.
